##

Интерактивное веб-приложение для расчета дебита скважины во времени после запуска при постоянном забойном давлении.

## Возможности

- Интерактивный ввод параметров
- Визуализация давления и дебита
- Экспорт результатов в PDF

## Технологии

- Python 3.14
- C++
- jupyter notebook
- Mercury
- Matplotlib
- NumPy, SciPy
