##

Интерактивное веб-приложение для расчета дебита скважины во времени после запуска при постоянном забойном давлении.

## Возможности

- Высокая производительность - расчетное ядро на C++
- Веб-интерфейс - интуитивный UI через Mercury
- Визуализация - графики давления и дебита в реальном времени
- Экспорт - сохранение результатов в PDF

## Технологии

- Backend: C++ 11 (расчетное ядро)
- Web Framework: Mercury (Python)
- Визуализация: Matplotlib, NumPy
- Платформа: Jupyter Notebook

## Запуск приложения

1. Убедитесь, что установлен Python и необходимые пакеты
2. Запустите команду в папке проекта:
   mercury run notebooks/app.ipynb
3. Откроется веб-браузер с интерфейсом приложения

## Технические детали

Расчетное ядро реализовано на C++ и компилируется в исполняемый файл (core.exe),
что обеспечивает значительное ускорение математических вычислений по сравнению
с Python реализацией.
Веб-интерфейс построен на основе Mercury, который автоматически преобразует
Jupyter notebook в интерактивное веб-приложение.

Разработано для учебных и исследовательских целей
